# Animation Brief — Nazia8Promo.kz + sos-sales.com

## Goal
Create two related motion systems that feel like one ecosystem.

- Nazia8Promo.kz = academy + implementation + operating logic
- sos-sales.com = product-first environment where the logic lives

---

## 1. Animation for Nazia8Promo.kz

### Purpose
Show how sales chaos becomes an operating system.

### Narrative
1. Incoming leads and WhatsApp messages are fragmented
2. There is no ownership and no clear next step
3. Roles appear: MOP / ROP / control
4. Workflow becomes structured
5. Final state: S:O:S Sales as Sales Operating System

### Visual objects
- WhatsApp chat badges
- lead cards
- owner tags
- next-step markers
- role labels
- structured workflow nodes

### Mood
- premium
- controlled
- strategic
- not flashy
- not startup-hyperactive

### Output recommendations
- 1600x900
- dark background
- mp4/webm or lottie
- 8–14 seconds seamless loop
- works muted
- should also work as a static poster frame

---

## 2. Animation for sos-sales.com

### Purpose
Show the real product workflow.

### Narrative
1. New lead enters Kanban
2. Lead is claimed by manager
3. Lead moves to qualification
4. Ownership and next step become visible
5. One lead goes to deal
6. Another goes to refusal
7. Mini-analytics pulse appears

### Visual objects
- Kanban columns
- lead cards
- WhatsApp badge
- masked phone
- owner chip
- stage chip
- next-step badge
- won / lost state
- micro analytics

### Must feel connected to Nazia8Promo animation
Use same:
- lead-card style
- badge style
- owner-chip style
- stage color logic
- navy + cobalt + amber palette

---

## 3. Shared visual system

### Palette
- Midnight navy: #081733
- Cobalt blue: #2D66F6
- Amber: #C88A35
- Warm ivory: #F5F0E8

### Typography
- one family only
- geometric / modern sans
- premium and clear

### Motion style
- smooth
- magnetic
- no chaotic bouncing
- no cheap gimmicks
- strong easing
- soft glow only where useful

---

## 4. Important message discipline

### Nazia8Promo
Do not sell "CRM first".
Sell:
- training
- implementation
- operating discipline
- system logic

### sos-sales.com
Do not oversell "autonomous AI".
Sell:
- WhatsApp-first workflow
- Kanban
- ownership
- role-based workflow
- practical visibility
- AI-assisted layer
