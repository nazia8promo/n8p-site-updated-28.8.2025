# Animation Integration Notes — Nazia8Promo.kz and sos-sales.com

## 1. Nazia8Promo.kz animation
Purpose:
- show sales chaos becoming an operating system

Narrative:
1. incoming leads and WhatsApp messages are fragmented
2. there is no ownership and no clear next step
3. roles appear: MOP / ROP / control
4. workflow becomes structured
5. final state: S:O:S Sales as Sales Operating System

Placement:
- homepage
- directly after hero
- inside current motion placeholder block

Recommended format:
- 1600x900
- dark background
- seamless loop
- mp4 / webm / lottie
- should work muted

Core message:
- not CRM-first
- operating logic first
- chaos -> roles -> ownership -> workflow -> S:O:S Sales

---

## 2. sos-sales.com animation
Purpose:
- show the real product workflow

Narrative:
1. new lead enters Kanban
2. lead is claimed by manager
3. lead moves to qualification
4. ownership and next step become visible
5. one lead goes to deal
6. another goes to refusal
7. mini-analytics pulse appears

Placement:
- homepage hero or first product section
- dominant visual block

Recommended format:
- 1600x900
- dark SaaS-style background
- seamless loop
- mp4 / webm / lottie

Core message:
- WhatsApp-first workflow
- Kanban
- ownership
- role-based workflow
- practical visibility
- AI-assisted layer

---

## 3. Shared visual DNA for both animations
Use the same:
- lead card style
- WhatsApp badge style
- owner chip style
- masked phone pattern
- stage chip logic
- navy + cobalt + amber palette

Shared palette:
- Midnight navy: #081733
- Cobalt blue: #2D66F6
- Amber: #C88A35
- Warm ivory: #F5F0E8

Typography:
- one sans family
- premium / clear / geometric

Motion style:
- smooth
- structured
- no chaotic bounce
- no cheap glitch effects
- no fake “AI fully autonomous” feeling
